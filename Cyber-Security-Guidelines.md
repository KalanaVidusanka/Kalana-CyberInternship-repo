What are common cyber security threats in a remote work environment? 

- Phishing - As the work is remote based, fake emails,calls or SMS can trick an user to click on malicious links or attachments that would leak their credentials or give access to their network infrastructure easily because they tend to trust the email as there is no physical interaction at remote environments mostly to justify if this is real or not. Moreover, nowadays phishing attacks have become advanced where they try to reduce the number of errors visible as well. 
- Lack of Broadband Security - As remote work is fully conducted via a Wi-Fi connection in most cases or celluar data sometimes, if the connection is not secure or there are no any protocols or controls intergrated it would make the conmpany network visible for the attacks easily to hack into. 
- Public Wi-Fi - In general working or dealing with confidential information or systems is not ethical as public Wi-Fi is a great gateway for hackers to gain access as the network traffic is not encrypted which makes hackers understand what is transmitted easily. 
- Outdated Devices - In some cases, if the working device such as laptop or phone is outdated it can also be vulnerable for attacks as it is not updated with latest security protections. 
- Easy Passwords - If the user is using a simple password and the same password across all accounts it can easily be cracked by a hacker.
- External Devices - In some cases, plugging unknown pen drives or CDs can install automatic viruses that can lead to serious attacks for the data files and the system. 
- Unknown applications - By downloading applications via the internet which could be cracked apps or any other applications can lead to automatic attacks that lead to ransomware attacks where all the files in the system could get locked and encryted which can only be decrypted after paying the relevant amount for the attacker which is also not a good thing. 
- Sharing tools, information - In some cases, if an user share their account or a platform data such as meeting notes or any reports which could go to wrong hands that leads to malicious planned attacks on these systems through the stats. 
- VPNs - In some cases, by using VPNs it can also give a hacker the chance to hack into your systems if the VPN tunnel is not secure enough. 
- File Sharing and Uploads - If the file sharing happens through less security protected platforms or the files are uploaded to random websites or AI for their personal gains it can also leave a trace for hackers to understand the systems and plan an attack. 

### What are best practices for keeping your devices and accounts secure?

- Strong Passwords & MFA - By creating unique long passwords with strong characters and combinations of letters each unique to each account will be good to avoid any credential theft or brute force attacks. Also setting up MFA can also add another security control so more ways should be conducted to proof the identity and give access to a new login device such as fingerprint, one time code etc. 
- Authenticator Apps - Authenticator apps also work similar to MFA where the website or the APP will be linked where a code will be generated in the app that will be used to login and prove the identity. 
- Avoiding Suspicious Emails - Always looking at the emails from a suspicious perspective could be good to not fall for phishing attacks. It is also good to communicate with the legit emails that had conversations from the day 1 to make sure that you received a real message from the same person or not. Also, it is best not to click links or attachments without looking for clues or errors. 
- Usauge of Apps or Tools - It is always good to mention and discuss the tools and apps that are being used to work on projects or tasks and also it is always good to confirm before downloading any new app or tool whether it will align with the company policies and whether it is legit application. 
- Updated Anti-Virus - Running daily scans and updating the security patches of the anti-virus softwares can make sure to detect any malicious behaviours in the system, downloadse etc. 
- Configure Home Wi-Fi Security - Investigate the Wi-Fi security passwords and change them to an unique long passwords with characters to make sure to avoid hackers brute forcing. Also check the relevant security controls such as protocols, licenses, Ip address etc. Also, make sure the encrytion protocol is an advanced protocol such as WPA2. 
- External Devices - Avoid Plugging in any unknown pen drives or CDs to the working devices to stop viruses spreading to the system and the network. 
- Software Updates - Keep all the devices such as Phones, Laptops updated with their latest OS updates as it releases security patches that can fight against attacks. 
- Device Lock - Lock all the devices such as laptops, phones with unique pins, patterns that can only be understood by yourself and also a max attempt limit to lock the device for sometime. It can make the stranger take a longer period of time to actually try out the password and after the failed attempts the device will be locked. 
- Browsing - Always check if the browsing websites have the "Paddlock Icon" and the "Digital Certificate SSL" to make the websites are secure. 
- Backups - Regularly backup all the data to a trusted external device or a cloud storage to make sure incase of data loss there will be an intial backup of data. 
- Permissions Limit - Review the types of permissioms accessed from different platforms and apps and if they seem to be suspicious it is better to not allow many permissions such as photos, videos etc.
- Monitoring - Always monitor the screen behaviors, any new documents,apps or files automatically installed in the system to see any suspicious behaviours. 

### Why is it important to lock your computer when away from your desk?

It is important to lock your computer when you are away as strangers or sometimes even peers who have the mindset of stealing or misusing your device data could easily get access to your device to transfer data or modify any confidential data. This can also happen in where attacker tend to use techniques like shoulder surfing where they would observe over your shoulder for pins, emails or anything and the moment you go away from the device without locking they can grant access. This applies to device theft as well as in case anyone grabs your devices and runs away when the device is not locked they can easily view your sensitive data. Due to this, it is a must to lock the device properly.

### What makes a strong password, and why should you use a password manager?

Nowadays, a strong password can be known as an unique password with 12-14 characters or more with a blend of symbols, numbers, upper and lowercase letters etc. As back in the day many websites tend to promote the max as 8 characters and it is happening in some websites and applications as well. However, I believe the more longer your password is with mix of everything makes it most safest account. In addition, you should always use passwords unique to every account and never repeat the same passwords for all accounts. 

Password manager is helpful to keep track/record of long passwords easily as nowadays android and apple offers autofill via the password manager as well which elimanates the time to memorize or writing down passwords somewhere. Passwords are safe in password manager as they tend to be highly encryted and they have the feature to randomly generate new passwords everytime with mix of many characters. Also, the autofill is done only by our actions such as biometric or a face unlock to confirm the autofill which makes it strong to not autofill on random websites or apps without our attention. 

### Why is two-factor authentication (2FA) important, and when should you enable it?

Two-factor authentication means additional security measure where there is two security measures to approve the identity and validate that you are the one who is signing up or accessing the data, account. In simple terms, there will be two security measures such as a pin along with biometric or SMS codes, authenticator app unlocks. This is very important and it can make sure to limit the access to unauthorized users accessing your data as even if they tend to gain one authentication solution they might not have the second security code or pin to access the system. This will prevent authorized access to your data and accounts. 2FA should always be activated for application accounts, Gmails, websites logins. This security measure should be done right after signing to a new account or setting one up. Nowadays, most application will remind you to set it up along with the registration.

After 2FA, there is the concept of Multi-Factor Authentication where it will require more than two identities which is the best security measure to adhere with as it is less likely to grant access to authorized users due to the series of proving identities.
